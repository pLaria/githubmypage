WhatsApp Sender

WhatsApp Sender es una herramienta que automatiza el envío de mensajes en WhatsApp Web.
Permite enviar mensajes a un número específico de teléfono con un mensaje personalizado, con la opción de repetir el envío múltiples veces.

Gestión de Usuarios con PHP y MySQL

Esta página web permite gestionar un registro de usuarios utilizando PHP y MySQL. Los usuarios pueden ingresar su nombre, apellido y correo electrónico a través de un formulario de registro. Los datos ingresados se almacenan en una base de datos MySQL y se muestran en una tabla en la misma página. Cada registro en la tabla tiene un botón de "Eliminar" que permite borrar registros específicos. La página también incluye estilos básicos y un diseño adaptado a dispositivos móviles.
